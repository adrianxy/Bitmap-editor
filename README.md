# Bitmap editor - semestr 1 - projekt 2

Program umożliwia następujące działania na plikach typu BMP:
- zmiana kolorystyki na odcienie szarości
- wyodrębnienie jednego koloru (red, green, blue)
- rozmycie całego obrazu
